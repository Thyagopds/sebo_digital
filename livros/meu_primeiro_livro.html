Meu Primeiro Livro de Programação

Autor: Nestor C. Burlamaqui

Preço: R$ 29,90

Gênero: Programação, Educação

Sinopse: Uma introdução prática à programação para estudantes iniciantes, apresentando os conceitos fundamentais de forma simples e acessível, utilizando exemplos e exercícios de fixação.

Detalhes: Este livro aborda os conceitos básicos de lógica de programação, variáveis, operadores, estruturas de decisão e repetição, com uma linguagem didática voltada para quem nunca programou antes. Ideal para primeiros cursos de programação no ensino médio ou superior.
